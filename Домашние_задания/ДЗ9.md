### Задание №1

На занятии мы начали проходить наследование классов:

**_Механизм, который позволяет одному классу (дочерний класс или подкласс) получать поля и методы другого класса (родительский класс или суперкласс)_**

---

За основу возьмем класс `Medkit` из предыдущего [**ДЗ №8**](./ДЗ8_Разбор.md) и для начала просто переименуем его в `Medicament` (_с англ. Лекарственное средство_)

```java
class Medicament {
    private int heal_points;
    private int count;

    Medicament(int heal_points, int count) {
        this.heal_points = heal_points;
        this.count = count;
    }

    public int use() {
        if (this.count > 0) {
            this.count--;
            return this.heal_points;
        } else {
            return 0;
        }
    }

    public void stat() {
        System.out.println("heal_points: " + heal_points);
        System.out.println("count: " + count);
    }
}
```

Теперь создадим еще 3 дочерних класса, которые наследуют свойства (поля и методы) класса `Medicament`. Пусть у каждого медикамента будет разное значение количества использования (`count`) и степень _отхила_ (`heal_points`):
- `FirstAidKit` (_с англ. Аптечка_)
  - `count` = 2
  - `heal_points` = 100
- `Adrenaline` (_с англ. Адреналин_)
  - `count` = 1
  - `heal_points` = 70
- `Painkillers` (_с англ. Обезболивающее_)
  - `count` = 10
  - `heal_points` = 20

**Чтобы класс был _"наследником"_ другого класса, после названия дочернего класса нужно указать ключевое слово `extends`, а потом название суперкласса**

Для того, чтобы задать значения `count` и `heal_points` для каждого подкласса напишем внутри каждого подкласса конструктор по умолчанию, внутри которого вызовем конструктор по умолчанию суперкласса (ключевое слово `super`) с заданными параметрами.

---

Разберем на примере подкласса `FirstAidKit`:

```java
class FirstAidKit extends Medicament {
    FirstAidKit() {
        super(100, 2);
    }
}
```

### Задание №2

Ключевое слово `final` - показывает что поле, метод или класс не может быть изменен:
- Поле - значение поля не может быть изменено после инициализации, т.е. значение задается один раз в конструкторе по умолчанию, в других методах его изменить будет нельзя.
- Метод - если мы наследуем класс, то методы в родительском классе помеченые `final` не могут быть переопределены, т.е. в дочернем классе нельзя изменить логику метода
  - На примере метода `use` в классе `Medicament`: пока этот метод не помечен ключевым словом `final`, в классе `FirstAidKit` можно прописать логику, что поле `count` увеличивается (получается бесконечная аптечка):

**_Этот код не нужно писать, просто пример:_**
```java
class FirstAidKit extends Medicament {
    FirstAidKit() {
        super(100, 2);
    }

    public int use() {
        if (this.count > 0) {
            this.count++;
            return this.heal_points;
        } else {
            return 0;
        }
    }
}
```

- Класс - если пометить класс ключевым словом `final`, его нельзя будет унаследовать

---

Т.к. логика наших медикаментов уже описана в классе `Medicament`, то пометьте ключевым словом `final` методы `use` и `stat`

---

На примере метода `stat`:

```java
public final void stat() {
    System.out.println("heal_points: " + heal_points);
    System.out.println("count: " + count);
}
```

### Задание №3

Ключевое слово `static` - помеченные поля и методы становятся общими для всего класса, а не для конкретного экземпляра класса.

Не будем много времени уделять данному ключевому слову, поэтому допишите класс `Medicament`, чтобы получилось следующее:

```java
class Medicament {
    private int heal_points;
    private int count;
    private static int med_count = 0; // Допишите эту строчку

    protected Medicament(int heal_points, int count) {
        this.heal_points = heal_points;
        this.count = count;
        this.med_count++; // Допишите эту строчку
    }

    public int use() {
        if (this.count > 0) {
            this.count--;
            return this.heal_points;
        } else {
            return 0;
        }
    }

    public final void stat() {
        System.out.println("heal_points: " + heal_points);
        System.out.println("count: " + count);
    }

    // Допишите метод get_med_count
    public static void get_med_count() {
        System.out.println("med_count: " + med_count);
    }
}
```
