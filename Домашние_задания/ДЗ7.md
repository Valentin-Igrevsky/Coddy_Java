---

Создайте 2 глобальыне переменные:
- Целочисленную `health_points`
- Строку `player_name`

Переменной `health_points` сразу задайте значение `100`

Напоминаю, что глобальные переменные инициализируются внутри класса, но вне методов, т.е.

```java
public class Main {
    // Тут глобальные переменные
    public void main(String[] args) {

    }
}
```

---

Создайте 3 метод:
- `damage`
  - возвращает ничего (т.е. `void`)
  - принимает целочисленную переменную `health_points`
  - внутри метода уменьшить глобальную переменную `health_points` на локальную для метода переменную `health_points` :)
- `heal` - возвращает ничего, принимает целочисленную переменную `health_points`
  - возвращает ничего
  - принимает целочисленную переменную `health_points`
  - внутри метода увеличить глобальную переменную `health_points` на локальную для метода переменную `health_points`
- `player_stats`
  - возвращает ничего
  - ничего не принимает (т.е. нет параметров)
  - внутри метода вывести в консколь
    - глобальную переменную `health_points`
    - глобальную переменную `player_name`

Напоминаю, что глобальные переменные в методах изменяются или выводятся с помощью ключевого слова `this`

Пример метода `damage`:

```java
void damage(int health_points) {
    this.health_points -= health_points;
}
```

Чтобы вывести в консоль глобальную переменную нужно:

```java
System.out.println(this.health_points);
```

---

В методе `main`, который у вас уже есть:
- Задать значение глобальной переменной `player_name` (можете указать свое имя или игровой ник)
- Вызвать метод `damage` с указанием передаваемого в метод значения
- Вызвать метод `heal` с указанием передаваемого в метод значения
- Вызвать метод `player_stats` с указанием передаваемого в метод значения :)

Пример вызова метода `damage`:

```java
public void main(String[] args) {
    damage(63);
}
```
