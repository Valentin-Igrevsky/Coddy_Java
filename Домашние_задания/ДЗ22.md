### Задание №2

1) **Создание нового файла и добавление в git**

- В проекте в разделе src, создайте новый Java Class:

![img.png](img/hw22/img.png)

- Дайте классу название Platform - этот класс нам понадобится на занятии

![img_1.png](img/hw22/img_1.png)

- После создания, появится окно с вопросом "Нужно ли добавить созданный файл в git-систему" - выбираем "Add"

![img_2.png](img/hw22/img_2.png)

- В созданном классе напишите следующий код (прикладываю готовый, т.к. дз на работу с git и github):

```java
import java.awt.*;

class Platform {
    private double x;
    private double y;

    private static final int PIXEL_SIZE = 4;

    private final Color[][] SPRITE;

    private int currentFrame = 0;

    public Platform(double x, double y, String spritePath) {
        this.x = x;
        this.y = y;
        this.SPRITE = SpriteLoader.loadSprites(spritePath).getFirst();
    }

    public void draw(Graphics2D g2d) {
        for (int row = 0; row < this.SPRITE.length; row++) {
            for (int col = 0; col < this.SPRITE[row].length; col++) {

                if (this.SPRITE[row][col] != null) {
                    g2d.setColor(this.SPRITE[row][col]);

                    g2d.fillRect(
                            (int)x + col * PIXEL_SIZE,
                            (int)y + row * PIXEL_SIZE,
                            PIXEL_SIZE,
                            PIXEL_SIZE
                    );
                }
            }
        }
    }
}
```
