### Задание №2 - Разделить физические объекты и статические

Под физическими объектами понимаются объекты, которые должны как либо двигаться, т.е. у этих объектов есть параметры скорости по осям (`velX` и `velY`) и сила кравитации. Один такой объект у нас уже есть - `Player`, в дальнейшем будет `Enemy` (враг) и некоторые декоративные элементы.

Что необходимо сделать? У нас есть класс `GameObject`:

```java
import java.awt.*;

class GameObject {
    protected double x;
    protected double y;

    protected double velX;
    protected double velY;
    protected boolean onGround;

    protected static final int PIXEL_SIZE = 4;

    protected Color[][] SPRITE;

    public GameObject(double x, double y) {
        this.x = x;
        this.y = y;
    }

      public void update() {
        onGround = false;

        x += velX;

        velY += gravity;
        y += velY;
      }

    public Rectangle getBounds() {
        int width = SPRITE[0].length * PIXEL_SIZE;
        int height = SPRITE.length * PIXEL_SIZE;

        return new Rectangle((int)x, (int)y, width, height);
    }

    public void draw(Graphics2D g2d) {
        for (int row = 0; row < SPRITE.length; row++) {
            for (int col = 0; col < SPRITE[row].length; col++) {
                if (SPRITE[row][col] != null) {
                    g2d.setColor(SPRITE[row][col]);
                    g2d.fillRect(
                            (int) x + col * PIXEL_SIZE,
                            (int) y + row * PIXEL_SIZE,
                            PIXEL_SIZE,
                            PIXEL_SIZE
                    );
                }
            }
        }
    }
}
```

Из этого класса нужно вычленить параметры отвечающие за физику, а точнее:

- Поля:

```java
protected double velX;
protected double velY;
protected boolean onGround;
```

- Методы:

```java
public void update() {
    onGround = false;

    x += velX;

    velY += gravity;
    y += velY;
}
```

Создайте новый класс `PhysicsObject`, который наследуется от `GameObject` и перенесите туда все необходимое для работы физики:

```java
class PhysicsObject extends GameObject {
    protected double velX;
    protected double velY;

    protected boolean onGround;

    protected final double gravity = 0.5;

    public PhysicsObject(double x, double y) {
        super(x, y);
    }

    public void update() {
        onGround = false;

        x += velX;

        velY += gravity;
        y += velY;
    }
}
```

Добавьте методы `getVelX`, `getVelY`, `getOnGround`, `setVelX`, `setVelY`, `setOnGround` по аналогии с `getX` и `setX` из класса `GameObject`

3. У игрока много методов, которые уже реализованы в классе `PhysicsObject` или `GameObject`, поэтому нужно почистить класс `Player` от лишних методов и полей:

```java
class Player extends PhysicsObject {
    private final double jumpStrength = -12;
    private final double speedStrength = 4;

    private int health = 100;

    private final ArrayList<Color[][]> SPRITES;

    private int currentFrame = 0;
    private int tickFrame = 0;

    public Player(double x, double y) {
        super(x, y);
        SPRITES = SpriteLoader.loadSprites("player");
        SPRITE = SPRITES.getFirst();
    }

    private void updateAnimation() {
        if (velX != 0) {
            tickFrame++;
            if (tickFrame > 8) {
                currentFrame = (currentFrame + 1) % SPRITES.size();
                SPRITE = SPRITES.get(currentFrame);
                tickFrame = 0;
            }
        } else {
            currentFrame = 0;
            SPRITE = SPRITES.getFirst();
            tickFrame = 0;
        }
    }

    @Override
    public void update() {
        x += velX;
        velY += gravity;
        y += velY;
        updateAnimation();
    }

    public void moveLeft() {
        velX = -speedStrength;
    }

    public void moveRight() {
        velX = speedStrength;
    }

    public void stop() {
        velX = 0;
    }

    public void jump() {
        if (onGround) {
            velY = jumpStrength;
            onGround = false;
        }
    }

    void damage(int damage) {
        health -= damage;
    }

    void heal(int heal) {
        health += heal;
    }

    int getHealth() {
        return health;
    }
}
```
