## Введение

Контрольная работа нацелена на повторение материала пройденного в первом полугодии для того, чтобы ученикам было проще вспомнить материал после новогодних каникул, чтобы не возвращаться к нему при прохождении новых тем, требующих знаний первого полугодия.

---

Контрольная содержит следующие темы:

1) Переменные
2) Ввод и вывод данных через консоль
3) Конструкция if-else
4) Конструкция switch-case
5) Одномерные и двумерные массивы
6) Циклы for и while, операторы break и continue
7) Классы, поля классов и методы классов
8) Ключевые слова private, public, final
9) Экземпляры классов
10) Интерфейсы
11) Перечисление enum

Темы объединены и распределены по 11 заданиям.

---

Для удобства выполнения работы все задания, необходимые входные данные и части кода описаны в одном файле.

Класс Main содержит вспомогательные методы для ускоренной проверки выполнения работы:
```java
static void task_1();
static void task_2(int health);
static void task_3(int level);
static void task_45();
static void task_67();
static void task_8();
static void task_910();
```

- Задания 1, 2, 3 и 8 выполняются внутри методов `task_1`, `task_2`, `task_3` и `task_8` соответственно
- Задания 4 и 5 внутри метода `task_45`
- Задания 6 и 7 внутри метода `task_67`
- Задание 9:
  - Пункт 1 выполняется вне класса Main
  - Пункт 2 выполняется внутри метода `task_910()`
- Задание 10:
  - Условие задания описано внутри метода `task_910()`
  - Само задание выполняется вне класса Main
- Задание 11 выполняется вне класса Main

Метод `public static void main(String[] args)` используется для автоматического запуска всех тестовых методов - этот метод как либо изменять не нужно!

Там, где в коде есть комментарий с троеточием (`// ...`) означает, что здесь нужно самостоятельно написать код по заданию

class Player {

    public Player(String name, int health) {
        this.name = name;

            this.health = MAX_HEALTH;

            this.health = 0;

    private void reset_health() {
        this.health = MAX_HEALTH;
    }
}
